# рџЋ¬ AI Video Assistant Agent

AI Video Assistant is a powerful, containerized Streamlit application that automates meeting intelligence. It allows users to upload local meeting recordings or submit YouTube URLs, transcribes the speech, extracts key insights (titles, summaries, action items, key decisions, and open questions), and establishes a Retrieval-Augmented Generation (RAG) chat engine to converse directly with the meeting content.

---

## рџљЂ Key Features

*   **Audio/Video Processing:** Supports local files and YouTube link downloads via `yt-dlp` and `pydub`.
*   **Local Speech-to-Text:** Converts audio into text using OpenAI's **Whisper** model locally.
*   **Intelligent Summarization:** Groups transcript segments and compiles bulleted final summaries via **LangChain** and **Mistral AI**.
*   **Insight Extraction:** Automatically extracts structured action items, owner tasks, deadlines, key decisions, and open follow-up questions.
*   **Contextual RAG Chat:** Builds a local **Chroma** vector DB of meeting text using HuggingFace embeddings (`all-MiniLM-L6-v2`) and provides an interactive chat interface to query the transcript contextually.
*   **Modern Aesthetics:** Dark-themed UI with glassmorphism effects, live pipeline progress widgets, and styled chat bubbles.

## рџ“‹ Prerequisites

Before running the application, ensure you have:
1.  **API Keys:**

    *   *Optional:* Sarvam API Key (if configured).
2.  **FFmpeg:**

    *   If running locally on Windows and `ffmpeg` is not in your system PATH, you can set `FFMPEG_PATH` in your `.env` file pointing to the `bin` directory of your FFmpeg installation (e.g., `C:\ffmpeg\bin`).

## вљЎ Setup & Run Instructions

Choose one of the following deployment options:

### Option A: Using Local Run Scripts (Recommended for Local Dev)

The project includes launcher scripts that automatically handle Python virtual environment creation, package installation, and execution:

*   **On Windows:**
    Double-click or run [run.bat](file:///c:/Users/shali/Downloads/GenAI/ForGit/GenAi-Projects/Ai-Video-Assistant-Agent/run.bat) from your command prompt:


*   **On Linux/macOS:**
    Give execution permissions and run [run.sh](file:///c:/Users/shali/Downloads/GenAI/ForGit/GenAi-Projects/Ai-Video-Assistant-Agent/run.sh):
    ```bash
    chmod +x run.sh
    ./run.sh
    ```

---

### Option B: Using Docker Compose (Recommended for Deployment)

This option encapsulates all dependenciesвЂ”including Python, packages, and **FFmpeg**вЂ”into a lightweight, reproducible container.

1.  Make sure you have [Docker](https://www.docker.com/) and Docker Compose installed.
2.  Populate your `.env` file in the root directory.
3.  Build and start the application container:
    ```bash
    docker-compose up --build
    ```
4.  Open your browser and navigate to `http://localhost:8501`.

*Note: The volumes mounted inside the `docker-compose.yml` ensure that local database states (`vector_db`) and downloaded media (`downloads`) persist across container restarts.*

---

### Option C: Manual Installation

If you prefer to run manual commands:

1.  Create and activate a python environment:
    ```bash
    python -m venv .venv
    # Windows:
    .venv\Scripts\activate
    # macOS/Linux:
    source .venv/bin/activate
    ```
2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3.  Run the Streamlit application:
    ```bash
    streamlit run app.py
    ```
